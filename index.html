Project
Proposal

Our project proposal consists of creating a webscraper that scrapes the stock data based on the ticker symbol and inserts it into a MongoDB database. In addition, we will also use an API to gather news articles, pulling article title, urls and author based on the stocks we chose. Finally, we will add them to a website and create pages for each symbol using Flask.
